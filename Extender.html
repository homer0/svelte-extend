generate(sfc, maxDepthopt) → {SFCData}

Source:

Takes an SFC data object, check if it extends from another and then does a recursive merge in order to generate a final SFC data object. It's recursive in case an SFC extends from an SFC that then extends from another...

How many components can be extended. For example, if a file extends from one that extends from another and the parameter is set to 1, the parsing will fail.
