    svelteExtendWebpackLoader(source: String)

    The loader that implements SvelteExtend for webpack.

    Static Public

    public svelteExtendWebpackLoader(source: String) source 

    import svelteExtendWebpackLoader from 'svelte-extend/src/webpack.js'

    The loader that implements SvelteExtend for webpack. The only option the loader has is allowedMaxDepth: How many components level can be extended. For example, if a file extends from one that extends from another and this is set to 1, the parsing will fail. This loader is async.
